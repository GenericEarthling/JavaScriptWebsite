Project 3 Menu

Functions, Methods, and Objects

  • Script 1

    • Mileage: The user inputs miles driven and the gallons of gas used. The script will calculate miles per gallon.

  • Script 2

    • Credit Limit: The user inputs credit information. The script will calculate new balance, and determine if user is over credit limit.

  • Script 3

    • Commission: The user inputs unlimited amount of sales items. The script will calculate commission on items and add it to the salespersons earnings.

  • Script 4

    • Payday: The user inputs employee's ID, hours worked, and pay rate for three employees. The script will calculate regular pay and overtime pay if applicable.

  • Script 5

    • Largest number: The user inputs 10 numbers. The script will determine which number is the largest.

  • Script 6

    • Palindrome: The user will input five numbers. The script will determine if the number is the same backwards as it is forwards.

  • Script 7

    • Encryption: The user will input four numbers. The script will encrypt the number then display the original number and the encrypted number.

  • Script 8

    • Decryption: This script will undo Script 7.
